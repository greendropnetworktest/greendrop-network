Frequently Asked Questions

Here you can find answers to the most common questions about Greendrop Network, its system, security, and reward structure.

About the System

What is Greendrop Network?

Greendrop Network is a global ecosystem that rewards users with digital incentives for contributing to environmentally friendly actions.

How does the system work?

Users participate in eco-friendly actions within the system. These contributions are digitally tracked and rewarded. For more details, please visit the System page.

Token & Reward Structure

Is this a cryptocurrency?

No. Greendrop tokens are not a public cryptocurrency. They are designed for use within a closed ecosystem.

What can earned tokens be used for?

Tokens can be used for in-system benefits, upcoming features, and community-based reward mechanisms.

Security & Privacy

Is the system secure?

Yes. Greendrop Network uses modern security standards, token-based authentication, and encrypted communication protocols.

Greendrop Network is currently in open beta. Digital rewards are not investment instruments.
